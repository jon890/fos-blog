# drizzle

## Purpose
Auto-generated Drizzle ORM migration artifacts. **Do not edit manually** — these files are generated by `drizzle-kit generate` and applied via `drizzle-kit push` or `drizzle-kit migrate`.

## Key Files

| File | Description |
|------|-------------|
| `0000_next_smiling_tiger.sql` | Initial migration SQL — creates all tables, indexes, and constraints |
| `meta/_journal.json` | Migration journal — tracks which migrations have been applied |
| `meta/0000_snapshot.json` | Schema snapshot at migration 0000 — used by drizzle-kit for diff calculation |

## For AI Agents

### Working In This Directory
- **Never edit these files directly** — always modify `db/schema/` files and regenerate
- To add a migration: modify `db/schema/*.ts` → `pnpm db:generate` → `pnpm db:push`
- The `meta/` folder is used internally by drizzle-kit for schema diffing

### Common Patterns
```bash
# Generate a new migration after schema change:
pnpm db:generate

# Apply migrations to DB:
pnpm db:push
# or:
pnpm db:migrate
```

## Dependencies

### External
- `drizzle-kit` — generates and manages migration files
